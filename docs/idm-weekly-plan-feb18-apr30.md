# IDM Enablement Plan (Week-by-Week)

**Program window:** Feb 18, 2026 → Apr 30, 2026  
**Launch requirements:**
- **Training Center live:** April 1
- **Timed races live:** final week of April

## Weekly Snapshot (1-line each)
- **Feb 18:** Lock scope, course definitions, success metrics, and start doc inventory.
- **Feb 24:** Ship Training Center foundation and continue doc-outline planning.
- **Mar 3:** Complete validation engine and finalize documentation ownership list.
- **Mar 10:** Launch timed-run capture and open coworker documentation signups.
- **Mar 17:** Ship leaderboards/medals and begin active article drafting.
- **Mar 24:** Run Tier 2 pilot, fix blockers, and review docs progress.
- **Mar 31:** Go live with Training Center on April 1; docs submission window opens.
- **Apr 7:** Optimize simulator quality and continue article submissions.
- **Apr 14:** Finalize IDM Olympics rules, dry run, and mid-April docs checkpoint.
- **Apr 21:** Lock competition build, launch comms, and final docs push.
- **Apr 28:** Run timed races, publish standings, and close out April docs submissions.

## Detailed Weekly Plan

## Week of Feb 18
**Focus:** Scope lock + success criteria + documentation inventory
- Finalize v1 scope: Training Center + Timed Evaluation + Leaderboard
- Finalize courses: Credentials, OUs, Groups
- Finalize pass/fail rules per course
- Start documentation planning: list all IDM articles/guides needed

**Deliverables**
- Approved scope doc
- Course definitions + validation criteria
- KPI list (participation, completion, best time)
- Initial documentation backlog (article title, audience, owner status)

**Measurable result**
- 100% of v1 courses have explicit pass criteria and owner
- 100% of target documentation topics captured in backlog

---

## Week of Feb 24
**Focus:** Training Center foundation + documentation outline planning
- Add Training Center entry point in portal
- Build guided practice flow for Credentials/OUs/Groups
- Persist completion status per user
- Convert documentation backlog into outlines (sections, screenshots needed, reviewer)

**Deliverables**
- Training Center shell + 3 guided courses
- Progress state persistence
- Documentation outline board ready for assignments

**Measurable result**
- Internal users can complete all 3 guided courses end-to-end
- 80%+ of planned docs have a draft outline and review owner

---

## Week of Mar 3
**Focus:** Validation engine + docs ownership readiness
- Implement deterministic completion checks for each course

- Add event logging for attempts/completions

**Deliverables**

**Measurable result**
- 100% of courses return deterministic pass/fail in QA
